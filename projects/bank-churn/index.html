European Bank — Churn/Inactivity Prediction

Problem

Identify at‑risk customers and provide actionable retention lists with thresholds and calibration.

Data

Tabular customer behaviour data (~10k rows).

Approach

Feature engineering; multiple classifiers; threshold tuning; simple calibration guidance for business.

Results

Core metrics
Up to 85% accuracy on hold‑out; ranked retention segments.
Business impact
Prioritised outreach to high‑risk customers; expected lift via targeted offers.

Artifacts

Code repository · Notebook / report

What I’d improve next

  • Run A/B or geo‑holdout where feasible; track lift and guardrails.
  • Add monitoring for drift, fairness, and alerting on KPI regressions.
