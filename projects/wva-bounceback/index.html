World Vision Australia — BounceBack Campaign Optimisation

Problem

Replace static donation asks with personalised, data‑driven amounts and tune contact frequency to reduce fatigue while improving conversion.

Data

De‑identified donation history, engagement signals, and supporter attributes.

Approach

Feature engineering; models: XGBoost/Random Forest; clustering for segments; rules for decisioning; role‑based dashboards for non‑technical users.

Results

Core metrics
Capacity‑Fit model (MAE ≈ $5.09, R² ≈ 0.31); Contact‑Saturation model (MAE ≈ $1.83, R² ≈ 0.857).
Business impact
Estimated response uplift 15–20% with controlled contact saturation; clearer targeting and reporting.

Artifacts

Code repository · Notebook / report

What I’d improve next

  • Run A/B or geo‑holdout where feasible; track lift and guardrails.
  • Add monitoring for drift, fairness, and alerting on KPI regressions.
